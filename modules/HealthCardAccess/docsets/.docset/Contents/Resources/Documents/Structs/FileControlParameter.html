    0 0 0 0 0 0 0 0 No information given
    0 0 0 0 0 0 0 1 Created
    0 0 0 0 0 0 1 1 Initialized
    0 0 0 0 0 1 - 1 Active
    0 0 0 0 0 1 - 0 Deactivated
