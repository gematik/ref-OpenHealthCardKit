  • readSelectedFile(expected:failOnEndOfFileWarning:offset:)

    Read the current selected DF/EF File

    Note

    This Publisher keeps reading till the received number of bytes is size or the channel returns 0x6282: endOfFileWarning. When the current channel maxResponseLength is less than the expected size, the file is read in chunks and returned as a whole.

    Throws

    Emits ReadError on the Publisher in case of failure.

    Declaration

    Swift

    public func readSelectedFile(expected size: Int?, failOnEndOfFileWarning: Bool = true, offset: Int = 0)
    -> AnyPublisher<Data, Error>

    Parameters

    size

    The expected file size. must be greater than 0 or nil. Note that failOnEndOfFileWarning must be false for this operation to succeed when size = nil.
    failOnEndOfFileWarning

    whether the operation must execute ‘clean’ or till the end-of-file warning. [default: true]

    Return Value

    Publisher that reads the current selected file
